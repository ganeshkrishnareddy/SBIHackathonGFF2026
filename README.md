# Sarathi AI — Agentic Customer Acquisition & Onboarding Concierge for SBI

Agentic Customer Acquisition & Onboarding Concierge for SBI, featuring multi-agent lead qualification, product matching, and compliance guardrails.

**GitHub:** [https://github.com/ganeshkrishnareddy/SBIHackathonGFF2026](https://github.com/ganeshkrishnareddy/SBIHackathonGFF2026) | **AI Studio:** [https://ai.studio/apps/6a4ec47a-48e0-4e2d-9294-a93d6b3f12f3](https://ai.studio/apps/6a4ec47a-48e0-4e2d-9294-a93d6b3f12f3)

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
